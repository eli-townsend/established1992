Project Description

The task for odfl.com was straight forward. We had to modernize the website along with applications and build it on a responsive framework. We went through many iterations of design critiquing layout, changing colors and textures to business approval. We created the new odfl.com to be mobile first development and were able to simplify the process on numerous applications by removing steps to get to the success criteria.
