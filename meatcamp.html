Project Description

For one of my freelance jobs, I was approached by Meat Camp Baptist Church to create a website for the church that allowed the members to keep up with updates, closings and cancellations. Since I would not be updating the website with all of the notifications, I decided that I would build the site on a content management system (CMS). After researching a few options, I chose WordPress because it would be easy to train members from the church on how to make updates, add content, and add users.

Tools

WordPress, HTML, CSS, PHP
