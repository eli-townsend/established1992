Project Description

Foundation is a responsive framework created by the web agency Zurb. When Foundation 6 was released, there was not a migration guide released with it at the time. Since I used Foundation in numerous projects, I decided that I would create a resource for designers and developers to showcase all of the changes between version 5 and version 6. In this resource, I decided on the best way to display the changes between each version is to have tabs labeled with the version of Foundation and highlight the code that was either added or removed from the element. For my work on this resource, Foundation recognized my work and gave me a shoutout! Visit the site or fork the project on Github.
