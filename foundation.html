Project Description

In my opinion, I feel that Zurb Foundation Framework is the best responsive to build a website or web application with. When Foundation 6 came out, there was not a migration guide created by Zurb so I took it upon myself to create this reference guide to display exactly what changed with between Foundation 5 and 6. Everything in the page besides the search is built with Foundation. Foundation even recognized me and gave me a shout out for my work. Definitely one of my finer moments as a designer!
