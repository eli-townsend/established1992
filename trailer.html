Project Description

Mission: Create an application for drivers to be able to lock/unlock and open/close their truck doors
When designing this app, I wanted to use the users' gestures to open, close, lock and unlock the door(s). Once the user lands on the door controls page, the user is able to press and hold on the specific door or entire truck which results in a context menu appearing. The context menu contains all of the options and is changed dynamically based on door/truck state. To select the option from the menu, the user will drag their finger on to the option and release their finger from the screen.
