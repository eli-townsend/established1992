Project Description

Transactional is a fictional company I created to depict the real world scenario I was put in. At my current employer, I lead a project for an application that would show the user the entire lifecycle of a PRO#. Due to NDA (Non-Disclosure Agreement), I cannot go over the full app in the design, but I want to focus on a key design element. This application is used by every department in the company which already puts a strain on the design process. The issue with the older version of the app was that there was not a good way to change from step to step. The main key feature that was requested was the ability to easily transition from one step in the process to next and the ability to go back. In the ideation phase of the design process, I presented the idea of using a horizontal timeline to show each step/change in the lifecycle. Different groups wanted different features, but the horizontal timeline was one solution because you could use the mouse to change points but also able to use the arrow keys to change points. We also created a active tooltip to label which step you are in the process. Another piece of functionality is the hover effect tooltip for non-active points. This showed the user which step was next or previous but gave them the ability to jump multiple steps.
